#### 笔记内容排版

这个笔记主要分为五个章节

* 基础算法
* 基础数据结构
* 进阶算法
* 进阶数据结构
* Leetcode专题

这里的前面四个章节主要是从算法和数据结构的角度去学习和记录不同算法和数据结构的特点以及应用场景，然后以leetcode题为sample去进一步阐释。

第五个章节尝试以Leetcode的一些topic（比如KSum）为切入点记录具体的一些有意思的Leetcode题目和套路。
